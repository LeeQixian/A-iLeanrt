# Unemployment

## 一、 上节课回顾与补充：储蓄、投资与金融体系

### 1. 挤出效应 (Crowding Out)
- **模型回顾**：储蓄 = 投资 ($S = I$)。
- **政府赤字的影响**：
    - 当政府增加支出 ($G$) 但不增加税收 ($T$) 时，公共储蓄 ($T-G$) 减少。
    - 这导致国民总储蓄（可贷资金供给）减少，供给曲线左移。
    - **结果**：利率上升，私人投资 ($I$) 减少。
    - **定义**：政府借款挤出了本来可以用于私人投资的资金。
- **中国视角的“挤出”**：
    - “国进民退”现象：2008年金融危机后，资金大量流向政府和国有企业（基础设施、国企扩张），导致民营企业（如温州企业）借贷困难，利率上升。
    - 中美差异：美国政府借款多用于消费性支出（医疗、养老），而中国政府借款本身往往包含大量投资（基建）。

### 2. 公式变形的理解
- **等式**：$Y - C - G = I$ (国民储蓄 = 投资)
- **变形**：$Y - C - T = I + (G - T)$
    - 左边：私人储蓄。
    - 右边：私人投资 + 政府赤字（政府借款）。
    - **含义**：如果是赤字增加，可以理解为总储蓄减少（供给左移），也可以理解为对资金的需求增加（需求右移，包含政府借款需求）。虽然图形坐标轴定义不同，但对利率上升的结论是一致的。

## 二、 失业的宏观经济学框架

### 1. 失业与经济增长
- **生产函数视角**：$Y = A \times F(L, K, H, N)$。
    - 失业意味着劳动力 ($L$) 的实际投入减少（需乘以 $1-u$，u为失业率），直接导致人均产出下降。
    - 失业对个人是巨大的不幸，对国家意味着资源浪费（GDP损失）。

### 2. 失业的分类
- **自然失业率 (Natural Rate of Unemployment)**：
    - **定义**：长期中经济趋向的正常失业率，不会自行消失。
    - **意义**：反驳了古典经济学认为“市场总能出清、不存在长期失业”的观点。
- **周期性失业 (Cyclical Unemployment)**：
    - **定义**：由于经济波动（如衰退、萧条）引起的短期失业。
    - **案例**：08年危机时，沿海工厂订单减少，民工返乡；经济好转后回流。

## 三、 失业的衡量 (Measuring Unemployment)

### 1. 美国的统计方法 (BLS/CPS)
- **方法**：每月随机抽样调查约6万个家庭（当前人口调查 CPS）。
- **人群分类**（针对16岁以上成年人）：
    1.  **就业者 (Employed)**：有工作（全职或兼职）。
    2.  **失业者 (Unemployed)**：没有工作，**且正在寻找工作**（或等待重返岗位）。*注：经济学关注非自愿失业。*
    3.  **非劳动力 (Not in Labor Force)**：没有工作且**不**找工作（如全职学生、退休人员、家庭主妇）。

- **关键指标计算**：
    - **劳动力 (Labor Force)** = 就业者 + 失业者
    - **失业率 (Unemployment Rate)** = $\frac{\text{失业者}}{\text{劳动力}} \times 100\%$
    - **劳动参工率 (Labor Force Participation Rate)** = $\frac{\text{劳动力}}{\text{成年人口}} \times 100\%$

### 2. 衡量中的问题
- **丧失信心的工人 (Discouraged Workers)**：想工作但因找不到而放弃寻找，被归类为“非劳动力”。这导致**低估**了真实的失业状况。
- **虚假求职者**：为了领取失业救济金而假装在找工作（实际上不想工作），被归类为“失业者”。这导致**高估**了失业率。（案例：电影《珍爱 Precious》中为了骗保假装找工作的母亲）。

### 3. 失业时间的悖论
- **现象**：
    - 大多数人经历的失业都是**短期**的。
    - 但在任何一个时间点进行统计，观察到的失业者大多处于**长期**失业状态。
- **比喻**：产科病房。大部分产妇是自然分娩（住院短），但去病房参观时看到的多数是剖腹产产妇（住院长）。
- **结论**：失业问题主要是由少数长期失业者构成的。

### 4. 中国的失业统计现状

#### 一、 现状与问题

##### 1. 统计数据的来源
中国现有的失业统计主要通过三种方式获取信息，但各自存在局限：
- **国家统计局与劳动社会保障部**：基于**登记制度**。当人失业并去劳动保障部登记（为了领取救济金）时，才会被记录。
	- 长期稳定在4.1%左右，极其不准确。未包含农民工、未登记的城市人口、被“买断工龄”或下岗未登记的人员。
- **国家工商行政管理局**：针对私营企业和个体户。由于统计局体制主要通过层层上报（如教育部报高校、国资委报国企），无法覆盖私营部门，因此通过工商登记作为间接渠道。
- **人口普查/抽样调查**：每隔一定时间进行的调查。
	- 2005年起启动调查失业率，但很少公开数据，相比登记失业率波动更大，更真实。

##### 2. 官方数据的矛盾与重复计算（以2004年为例）
- **数据概览**：
    - 成年人口约10亿，劳动力（经济活动人口）约7.68亿，就业人口7.52亿。
    - 计算出的失业率极低（约2.1%），就业率高达75.7%（高于美国）。
- **统计缺陷**：
    - **高估就业（重复计算）**：农民工被计算了两次。在城市被单位算作就业，在农村户籍地又被算作农民就业。这大幅降低了账面上的失业率。
    - **范围不准确**：包含了非本国居民、退休返聘人员等。
    - **低估失业**：很多失业者并不去登记。

##### 3. 核心指标：城镇登记失业率
中国的官方指标是**城镇登记失业率**，存在显著缺陷：
- **范围限制**：只统计城镇，完全忽略农村失业（虽然发展中国家农村失业确实难界定）。
- **登记偏差**：依赖主动登记。很多人失业了不去登记，或者为了领低保没失业也去登记。
- **更科学的方法**：老师指出最好采用美国的**调查失业率**（抽样入户询问），而非登记制。

##### 4. 修正计算的尝试（“打补丁”算法）
为了矫正重复计算，统计上采用复杂的公式：
- **分子**：登记失业人数（可能低估）。
- **分母**：城镇单位就业人员 - 使用的农村劳动力 - 聘用的离退休人员 - 外方人员 + 不在岗职工 + 私营个体户就业人员。
    - *注：这种“打补丁”试图剔除重复计算，但私营个体户等数据依然难以准确获取（如卖羊肉串、开黑车等未登记就业）。*

#### 二、 历史变迁与结构分析

##### 1. 失业率的历史走势
- **70年代**：失业率非常高（知青返城找不到工作）。这也是中国改革开放初期鼓励个体户、私营经济发展的背景（被就业压力逼出来的）。
- **80年代至90年代中期**：失业率逐渐下降，维持在较低水平（约4.2%）。
- **90年代末**：由于国企改革导致“下岗潮”，失业率开始上升。

##### 2. 特殊群体：“下岗职工” (Xiagang)
- **定义模糊**：在中国，“下岗”和“失业”曾被区别对待。下岗职工往往不被计入官方失业率。
	- 1998-2005年间特殊的统计口径。
	- 进入“再就业中心”的不算失业，离开中心未就业的才算登记失业。2005年后该概念取消，并轨到失业统计中。
- **调整后的失业率**：如果在2004年把下岗职工计入失业，失业率会从4.2%升至**5.6%**（下岗与失业人数当时各占一半）。
- **消失的下岗职工**：
    - 数据现象：下岗职工人数迅速下降（从600万降至100万），但同期登记失业人数增加幅度较小（从600万增至800万），中间有约300万的缺口。
    - **去向推测**：
        1.  找到了工作。
        2.  **气馁的工人 (Discouraged Workers)**：退出了劳动力市场（例如回家带孩子、做家务）。老师以自家老人为例，说明这部分隐性失业未被统计。

##### 3. 失业者的构成（基于调查数据）
如果询问所有失业者“你是属于哪类失业？”，结果显示：
- **下岗或内退（~36%）**：比例最高，且可能低估（只有登记了才算）。
- **毕业后未找到工作（~20%）**：
    - **本科生**：在所有失业的本科生中，**52%**是因为毕业后未找到工作。
    - **研究生**：在所有失业的研究生中，**42.9%**是毕业未找到工作；但**辞职**比例高达28.6%（说明研究生对工作更挑剔）。
- **其他类型**：
    - 单位破产。
    - 合同期满（隐性辞退）。
    - **承包土地被征用**：农民被迫“城镇化”，虽转为城镇户口但未获安置工作。

## 四、 失业的原因 (Why is there Unemployment?)

为什么长期中存在自然失业率？（即劳动力市场不能完全出清）

### 1. 理论基准 (Benchmark)
- **经典理论**：在理想市场中，供求平衡，不存在非自愿失业。如果你不接受当前工资而不工作，属于“自愿失业”。
- **现实困境**：经典供求理论无法解释现实中的失业，因此需要引入新理论。
### 2. 摩擦性失业 (Frictional Unemployment)
- **定义**：由于工人和工作岗位匹配需要时间而引起的失业。通常是短期的。
- **成因**：
    - **寻找工作 (Job Search)**：信息不完全，匹配需要过程。因此可能同时存在“招工难”和“就业难”，
    - **部门转移 (Sectoral Shifts)**：产业结构调整（如农业向工业转移，夕阳产业向朝阳产业转移）、地区兴衰，工人需要重新培训和寻找机会。
	    * *案例*：底特律（汽车业衰退）vs 硅谷（IT业兴起）；中国农民工进城（从农村转移到城市需要搜寻时间）。
- **公共政策的影响**：
    - **就业服务机构**：提供信息，减少搜寻时间（降低失业）。
    - **培训计划**：帮助技能转换（降低失业）。
	    * *案例*：经济学家**茅于轼**曾开办学校，义务培训农民工如何适应城市生活（如怎么逛超市、怎么买东西），帮助其转移就业。
    - **失业保险 (Unemployment Insurance)**：
        - 负面：降低了工人找工作的紧迫感，增加了摩擦性失业的时间。
        - 正面：增加了匹配的质量（工人可以等待更适合的工作，而非被迫接受差工作），由于匹配度提高，长期生产率可能提高。
        * 类比：社会单身率上升不一定是坏事，可能意味着人们在追求更高质量的婚姻匹配。
### 3. 结构性失业 (Structural Unemployment)
- **定义**：由于工资高于均衡工资水平，导致劳动力供给大于需求，从而产生的失业。通常解释长期失业。
- **三大原因**：

    - **A. 最低工资法 (Minimum Wage Laws)**
        - 当法定最低工资高于市场均衡工资时，造成劳动力过剩（失业）。
        - 主要影响非熟练工人、青少年等低技能群体。
        - *注：在中国该因素影响相对较小，执行力度不一。*

    - **B. 工会 (Unions) 与集体谈判**
        - 工会作为卡特尔，通过罢工威胁将工资提高到均衡水平之上。
        - **局内人与局外人**：工会成员（局内人）获益，非工会成员（局外人）面临失业或工资降低。
        - *注：在中国工会主要职能不是工资谈判，西方国家工会力量也在下降。*

    - **C. 效率工资 (Efficiency Wages)**
        - **核心逻辑**：企业**自愿**支付高于市场均衡的工资，以提高生产效率。
        - **四种解释机制**：
            1.  **工人健康**：(发展中国家) 吃得好，干得动。
            2.  **工人流动率**：减少员工跳槽，降低培训成本（例如：飞行员培训费高昂，需高薪留人）。
            3.  **工人素质**：吸引更高能力的申请者（例如：投资银行高薪筛选人才）。
            4.  **工人努力程度**：解决**信息不对称**（偷懒难以监督）问题。如果工资很高，工人一旦被解雇的代价巨大（很难找到同等薪酬工作），因此会努力工作（例如：投行的高压环境）。

## 五、 课堂互动与讨论：中国的失业问题

### 1. 中国失业的主要类型
- **摩擦性/周期性混合**：
    - **城乡流动**：农民工在城市找不到工作返乡，或季节性失业。
    - **体制转型**：国企下岗（摩擦性），所有制结构调整带来的失业。
    - **大学生就业**：属于寻找工作带来的摩擦性失业（寻找匹配的岗位）。

### 2. 《劳动合同法》的讨论
- **背景**：规定连续签两次固定期限合同后，第三次必须签**无固定期限合同**（类似终身教职）。
- **对失业的影响（讨论观点）**：
    - **增加失业**：
        - 企业为了避免签无固定期限合同，会在第二次合同到期前解雇工人（缩短雇佣期），甚至让员工“先辞职、隔段时间再入职”以重置工龄。
	        - **增加了摩擦性失业**，使企业不敢长期雇佣。
        - 企业雇佣新人的门槛变高，不愿意招人，尤其是年轻人进入劳动力市场变难。
        - 案例
		    - **清华幼儿园案例**：为了规避第三个聘期的规定，采取了辞退重聘的操作。
		    - **华为案例**：曾出现大规模“辞职再上岗”事件，目的是中断工龄，将老员工变为新员工，规避法律责任。
    - **减少失业（对部分人）**：
        - 对于高素质、企业想留住的员工，工作稳定性增加，不再面临失业风险。
    - **结论**：改变了企业的激励机制，可能导致劳动力市场僵化，增加了特定群体（如新毕业生或低技能工人）的摩擦性或结构性失业风险。
- **总结**：经济学强调**人会对激励做出反应**。政策制定不仅要看直接意图，更要警惕间接的、意想不到的后果（如失业保险增加失业时长、劳动合同法增加解雇频率）。

## 六、 拓展材料与推荐
- **电影**：《珍爱》(Precious) —— 反映美国底层为了领救济金假装找工作的现象。
- **书籍**：《打工女孩》(Factory Girls) —— 描写中国农民工（尤其是女性）进城打工的真实生态，反映城乡流动的摩擦性失业。
- **概念辨析**：马克思的“产业后备军”理论与效率工资理论在解释高工资导致失业（维持一群找不到工作的人以激励在岗者）上有异曲同工之妙。
