

## 一、 金融体系概述
金融体系（Financial System）是经济中促使一个人的储蓄与另一个人的投资相匹配的一组机构。
- **核心功能**：**匹配（Match）**。
    * 将储蓄者（Savers）和借款者（Borrowers）匹配起来。
    * 区别于普通商品的“交易（Trade）”（同质化），金融匹配带有异质性（不同风险偏好、不同期限）。
    - **实质**：资源的转移。将资源从消费（吃蛋糕）转移到投资（造锤子/机器），从而改变实体经济的生产结构。
- **两大类机构**：
    1.  **金融市场（Financial Markets）**：储蓄者直接向借款者提供资金（如债券市场、股票市场）。
    2.  **金融中介（Financial Intermediaries）**：储蓄者通过中间人间接提供资金，储蓄者和借款者不见面（如银行、基金）。
### 1. 金融体系的作用
- **匹配 (Match)**：
- **实质**：**资源转移**。
    - **案例**：张三存钱不买蛋糕（减少消费），李四借钱买机器（增加投资）。实体经济中，资源从生产消费品转移到了生产资本品。
    - **注**：二手股票交易（炒股）如果只是所有权转手，不涉及资源转移，不是宏观关注的重点。宏观关注的是IPO或增发等涉及实体投资的行为。

### 1. 债券（Bonds）
- **定义**：债务证明书（IOU），即欠条。
- **特征**：
    - 期限（Term）。
    - 信用风险（Credit Risk）。
    - 税收待遇（Tax Treatment）：如美国市政债券免税。
- **中国现状**：债券市场相对不发达。主要是国库券（老百姓排队买），地方政府发债正在改革中（以前依赖土地财政），企业债较少。

### 2. 股票（Stocks）
- **定义**：代表企业的所有权（Equity Finance），区别于债务融资（Debt Finance）。
- **经济学核心概念：所有权（Ownership）**
    - **剩余索取权（Residual Claim）**：支付完工资、债息、租金后，剩下的利润归股东。好处是上限高（暴富），坏处是兜底风险。
    - **剩余控制权（Residual Control）**：合同未规定的事项由老板说了算。
    - **激励机制**：拥有剩余索取权的人必须拥有剩余控制权，否则激励不对称。
    - **案例对比**：
        - **企业**：权责相对清晰，效率高。
        - **学校**：不如企业清晰（老师可以说“没规定我必须上这门课”）。
        - **家庭**：市场上最糟糕的“企业”。谁是剩余索取者？孩子有出息都说是自己的功劳，出问题都赖别人，责权不清。
- **股票价格**：由需求决定（供给相对固定）。反映了人们对未来经济前景的预期。
    - **中国股市特殊性**：市场太小（只占资金的一小部分），投机性强，导致价格波动巨大，作为宏观经济晴雨表的功能要打折扣。

### 3. 核心指标：市盈率（P/E Ratio）
- **定义**：价格（P）/ 每股收益（E）。
- **经济学含义**：
    - 市盈率的倒数（E/P）近似于**股票的“利息率”**（收益率）。
    - P/E越高，意味着收益率越低（股票贵），或者市场预期未来增长极快。
- **案例数据**：
    - 美国典型P/E约为15（约6%回报率）。
    - 中国P/E常年较高（有时甚至50-60，回报率极低）。
- **中国高市盈率的原因**：**储蓄率太高，资金无处可去**。
    - 中国人连大蒜、绿豆都能炒，钱多导致资产价格（房价、股价）虚高。这反映了缺乏有效的实体创新投资渠道（“悲剧”）。

### 4. 其他金融概念
- **有效市场假说（Efficient Markets Hypothesis）**：
    - 资产价格反映了所有公开信息。
    - 结论：**不要试图通过看K线图或消息来打败市场**（除非你是内幕人 Insider）。专家炒股也不一定比猴子扔飞镖强，指数基金（Index Fund）通常优于管理型基金。
- **中国居民金融资产结构**：
    - **存款**占比极高（60%以上）。
    - 债券极少（除了国债买不到别的）。
    - 股票波动极大（投机性强，散户多）。

## 二、 宏观经济模型：储蓄与投资的决定

我们通过**国民收入恒等式**出发，构建一个**可贷资金市场（Market for Loanable Funds）**模型。

### 1. 从恒等式到模型
- **国民收入恒等式**：$Y = C + I + G + NX$。
- 假设**封闭经济**（$NX=0$）：$Y = C + I + G$。
- 变换公式：$Y - C - G = I$。
- 定义**国民储蓄（S）**：$S = Y - C - G$。
- **结论**：**$S = I$（储蓄等于投资）**。

### 2. 储蓄的分解
引入税收（T），将国民储蓄分解为两部分：
$$S = (Y - T - C) + (T - G)$$
1.  **私人储蓄（Private Saving）**：$Y - T - C$。家庭收入减去税收和消费后的剩余。
2.  **公共储蓄（Public Saving）**：$T - G$。政府税收减去政府支出。
    - $T > G$：预算盈余（Budget Surplus）。
    - $T < G$：预算赤字（Budget Deficit，即负储蓄）。

### 3. 可贷资金市场模型（Market for Loanable Funds）
这是一个为了宏观分析虚构的单一市场，将所有金融市场（股、债、银）加总。
- **供给方（Supply）**：**储蓄（Savings）**。来自家庭（主要）和政府（盈余时）。
- **需求方（Demand）**：**投资（Investment）**。来自企业（买设备）和家庭（买房）。
- **价格**：**实际利率（Real Interest Rate）**。
    - 这也是资金的价格。必须是实际利率，因为人们关心的是未来的购买力（实体经济的蛋糕），而不是名义上的钱。

### 4. 均衡机制
- **供给曲线**：向上倾斜。利率越高，当期消费的机会成本越高，人们越愿意储蓄。
- **需求曲线**：向下倾斜。利率越高，借款成本越高，企业投资意愿下降。
- **均衡**：利率调整使得 $S = I$。

## 三、 政策分析（比较静态分析）

通过移动供求曲线来分析政策对经济的影响。

### 政策 1：对储蓄激励（如利息税调整）
- **场景**：如果政府**减少利息税**（或对储蓄免税）。
- **分析**：
    1.  增加了储蓄的回报，**供给曲线右移**。
    2.  **结果**：均衡利率**下降**，投资数量**增加**。
- **结论**：鼓励储蓄的政策能降低资金成本，促进投资和增长。
- *注：如果是增加利息税，则反之。*

### 政策 2：对投资激励（如投资税收抵免）
- **场景**：企业购买设备可以抵扣增值税或获得税收优惠。
- **分析**：
    1.  增加了借款投资的意愿，**需求曲线右移**。
    2.  **结果**：均衡利率**上升**，储蓄/投资数量**增加**。
- **难点辨析**：
    - 为什么鼓励储蓄导致利率下降，鼓励投资导致利率上升？
    - 虽然结果都是投资量增加，但前者是资金多了（供给增），后者是大家抢资金（需求增）。

### 政策 3：政府预算赤字与盈余
- **场景**：政府支出增加（G上升）或税收减少（T下降），导致赤字。
- **分析（挤出效应 Crowding Out）**：
    1.  公共储蓄（$T-G$）减少，导致国民总储蓄减少。
    2.  **供给曲线左移**（约定俗成：政府借款视为减少国民储蓄，而不是增加需求）。
    3.  **结果**：均衡利率**上升**，私人投资**减少**。
- **含义**：政府为赤字融资（借钱）占据了额度，抬高了利率，**“挤出”**了私人的投资。
    - *中国案例*：4万亿刺激计划中，政府和国企借走了大量资金，可能导致民企更难借钱。
    - *美国案例*：长期赤字可能导致资本存量下降，影响长期增长（虽然美国靠借外国人的钱维持）。

## 四、 理论辨析：恒等式 vs. 均衡等式

课堂最后澄清了一个容易混淆的概念：
1.  **恒等式（Identity）**：
    - **定义**：通过定义或会计规则人为构造的相等（如：利润=收入-成本；$Y=C+I+G$）。
    - **性质**：永远成立，不需要理论解释，用于计算和核算。
2.  **均衡等式（Equilibrium Equation）**：
    - **定义**：描述经济运行机制达到平衡状态的等式（如：$Q_d = Q_s$；$S = I$ 在模型中）。
    - **性质**：通过机制（如价格/利率调节）实现。如果机制失灵，可能不相等（非均衡）。

**总结**：$S=I$ 最初是从国民收入恒等式推导出来的，但在宏观模型中，我们用它来描述**金融市场如何通过利率调节**达到均衡。如果有人问“为什么储蓄等于投资”，回答“因为这是恒等式”是技术含量低的回答；回答“因为金融市场的利率调节机制”才是经济学的回答。
